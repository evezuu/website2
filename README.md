# AP Environmental Science Website
This is a website that allows me to easily do review for my AP exam through its features.

## List of Features
Searchbar: Allows searching through the whole website for any words/keyterms you want to go back and look at or review.
Index: An index at the top of the website to help you jump to pages.
Table of Contents (vocab): A table of contents in the vocabulary tab for easy navigation

## Languages used
HTML, CSS, and JavaScript

## Things learned
I learned a lot about how to use JavaScript through all the features I made. I also learned quite a bit about how to make navigation guides & hideable text through html, and a lot about formatting pages in CSS.
